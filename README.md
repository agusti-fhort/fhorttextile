# FHORT Textile Tech

SaaS PLM tècnic per a moda. Gestió del cicle de vida tècnic des de design freeze fins a top of production.

## Stack
- Backend: Django REST Framework + django-tenants
- Frontend: React + Vite + Tailwind
- BD: PostgreSQL (schema-per-tenant)
- Cache/Jobs: Redis + Celery

## Estructura
- /backend — Django API
- /frontend — React SPA
